## Preparation

1. new ```gridworld.jar``` is required.

## Usage

1. ```ant compile```   
    Compile all the java files into the folders automaticaly layered
1. ```cd class```  
    Move to the class folder.
1. ```java info.gridworld.maze.MazeBugRunner```
    Run the runner.
